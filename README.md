## About
Welcome to **Lia's CSS Art Studio**, a creative website that showcases CSS as an art form and gives tips for aspiring designers. This project focues on responsive and interactive design. It also provides a fun and educational experience.

This website was created to display CSS art creations and provide helpful resources for anyone that is interested in not only web design, but design as a whole. It includes a responsive layout that work on desktop and mobile devices.

## Goals
- Inspire students to explore a different form of art/design.
- Provides a hands-on platform to experiment with more interative CSS.
- Help teach more ways for adding unique designs to webpages.
- Encourage learning through a different example.

## Target Audience
This website is designed for **middle school through college students** who:
- Are interested in web design.
- Want to learn/try something new.
- Enjoy working with interactive projects.
- Searching for different ways to strengthen their webpages.

## Features
- **Homepage:** Highlights featured CSS art projects.
- **Create Page:** Place to try out creating CSS art.
- **Tips Pages:** Offers guides and practice for CSS creation.
- **About Page:** Shares information about the online studio and creator.
- **Responsive Design:** Works seamlessly across desktops, tablets, and smartphones.
